# MUUD Health — Frontend (Flutter Web)

Minimal UI to demo:
- Register/Login (JWT)
- Journal: add entry (+ mood 1–5) & list
- Contacts: add contact & list

## Prereqs
- Flutter SDK installed (`flutter --version` works)
- Backend running at `http://localhost:4000`

## Run (web, Chrome)
```bash
cd frontend
flutter pub get
flutter run -d chrome
